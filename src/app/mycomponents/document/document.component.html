Bored Public API Angular Project

Prerequisites

To run this project, ensure that you have the following installed on your machine:

  • Node.js: https://nodejs.org
  • Angular CLI: Run npm install -g @angular/cli to install the Angular CLI globally.

Installation

  1. Clone the project from GitHub: https://github.com/Darshan9139/Board-API
  2. Navigate to the project directory using a terminal or command prompt.
  3. Install the project dependencies by running the following command:
    npm install
  4. After the dependencies are installed, start the development server by running the following command:
    ng serve
  5. Open your web browser and navigate to http://localhost:4200/ to access the application.

Usage

Once you have the project running, you can interact with the Bored Public API using the provided user interface.

  1. On the home page, you will see various options to select:
  • Activity Type: Choose the category of activity you are interested in, such as education, music, social, etc.
  • Participants: Select the number of participants for the activity, ranging from 1 to any number.
  • Get Activity: Click this button to retrieve an activity based on your selected options.
  • Get Random Activity: Click this button to retrieve a random activity without specifying any options.

After selecting your preferences and clicking either the "Get Activity" or "Get Random Activity" button, the application will make a request to the Bored Public API and display the retrieved activity on the page.

You can repeat the process by selecting different options or clicking the buttons again to fetch new activities.

Customization

If you wish to modify or enhance the project, you can make changes to the source code available in the project directory.

  • Components: The main logic and UI for the application can be found in the src/app/mycomponents directory.
  • Services: The API interaction and data retrieval are handled by services located in the src/app/myservices directory.

Feel free to explore and modify the code as per your requirements.

Conclusion

The Bored Public API Angular project allows you to interact with the Bored Public API to retrieve activities based on your preferences. Enjoy exploring various activities and have fun!
